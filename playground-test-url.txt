Manual Test Steps:
1. Go to https://playground.wordpress.net/
2. Click "Create new site"
3. Once WordPress loads, go to Plugins → Add New Plugin
4. Upload the seo-forge-plugin.zip file
5. Activate the plugin
6. Navigate to SEO Forge in the admin menu
7. Test the UI and features
