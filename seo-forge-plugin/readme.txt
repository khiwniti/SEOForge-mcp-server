== Description ==

SEO Forge is a comprehensive SEO plugin that provides AI-powered content generation, advanced SEO analysis, keyword research, and optimization tools. Built for universal use without any licensing restrictions.

**Key Features:**

* **AI Content Generator** - Generate SEO-optimized blog posts, product descriptions, and more
* **SEO Analyzer** - Comprehensive SEO analysis with actionable recommendations
* **Keyword Research** - Discover high-value keywords with search volume and difficulty data
* **Meta Box Integration** - Easy-to-use meta boxes for post editing
* **Multi-language Support** - Generate content in multiple languages
* **Flux Image Generation** - State-of-the-art Flux AI models for high-quality image generation
* **No License Required** - Free to use without restrictions

**Supported Content Types:**
* Blog Posts
* Product Descriptions
* Landing Pages
* How-to Guides
* News Articles
* Social Media Posts

**Supported Languages:**
* English
* Thai
* Spanish
* French
* German
* Italian
* Portuguese
* Russian
* Japanese
* Korean
* Chinese

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-forge` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the SEO Forge menu to configure your settings
4. Start generating content and optimizing your SEO!

== Frequently Asked Questions ==

= Do I need an API key? =

The plugin works with the default API endpoint, but you can configure your own API key for enhanced features.

= What languages are supported? =

SEO Forge supports content generation in 11 languages including English, Thai, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean, and Chinese.

= Can I generate images with content? =

Yes! SEO Forge can generate high-quality images using state-of-the-art Flux AI models. Supports multiple Flux variants including flux-schnell for fast generation and flux-dev for highest quality.
