A comprehensive WordPress SEO plugin with AI-powered content generation and intelligent customer service chatbot.

== Description ==

SEO-Forge is a powerful WordPress plugin that combines advanced SEO optimization with AI-powered content generation and an intelligent chatbot system. This plugin helps you create SEO-optimized content, analyze your website's performance, and provide automated customer support.

= Key Features =

* **AI-Powered Content Generation**: Generate high-quality, SEO-optimized content using advanced AI technology
* **Intelligent Chatbot**: Provide 24/7 customer support with an AI-powered chatbot
* **SEO Analysis**: Comprehensive SEO analysis and optimization recommendations
* **Keyword Research**: Advanced keyword research and suggestion tools
* **Meta Tag Optimization**: Automatic meta title and description generation
* **Content Optimization**: Real-time content optimization suggestions
* **Multi-language Support**: Support for multiple languages including English and Thai

= AI Content Generation =

* Generate blog posts, articles, and web content
* SEO-optimized title suggestions
* Meta description generation
* Keyword-rich content creation
* Image generation for content

= Chatbot Features =

* 24/7 automated customer support
* Customizable chatbot responses
* Integration with your website's design
* Multi-language support
* Analytics and conversation tracking

= SEO Tools =

* On-page SEO analysis
* Keyword density analysis
* Meta tag optimization
* Content readability scoring
* SEO recommendations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-forge` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the SEO-Forge menu in your WordPress admin to configure the plugin
4. Enter your API key in the settings to enable AI features
5. Configure the chatbot settings according to your needs

== Frequently Asked Questions ==

= Do I need an API key to use this plugin? =

Yes, you need an API key to access the AI-powered features including content generation and chatbot functionality. You can obtain an API key from the SEO-Forge website.

= Is the plugin compatible with other SEO plugins? =

SEO-Forge is designed to work alongside other SEO plugins, but for best results, we recommend using it as your primary SEO solution.

= Can I customize the chatbot appearance? =

Yes, the chatbot appearance can be customized through the admin settings to match your website's design.

= What languages are supported? =

Currently, the plugin supports English and Thai, with more languages planned for future releases.
