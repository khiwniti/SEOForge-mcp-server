üîß Integration Guide

Add our AI chatbot to your website in just 3 simple steps:

Step 1: Include the Widget Script

<script src="https://universal-mcp-server.onrender.com/static/chatbot-widget.js"></script>

Step 2: Initialize the Chatbot

<script> UMCPChatbot.init({ serverUrl: 'https://universal-mcp-server.onrender.com', websiteUrl: 'https://your-website.com', companyName: 'Your Company Name', primaryColor: '#667eea', position: 'bottom-right', autoOpen: false, showNotifications: true }); </script>

Step 3: Customize (Optional)

Available configuration options:

  • serverUrl: Your MCP server URL
  • websiteUrl: Your website URL for context
  • companyName: Display name in chat header
  • primaryColor: Brand color for the widget
  • position: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
  • autoOpen: Automatically open chat after delay
  • showNotifications: Show notification badges
