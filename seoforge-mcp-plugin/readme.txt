AI-powered SEO content generation and optimization plugin using MCP (Model Context Protocol) server.

== Description ==

SEOForge MCP Plugin connects your WordPress site to a powerful AI-driven SEO platform that helps you:

* **Generate SEO-optimized content** - Create blog posts, articles, and product descriptions
* **Analyze content performance** - Get real-time SEO scoring and recommendations
* **Research keywords** - Discover high-value keywords for your niche
* **Bilingual support** - Generate content in English and Thai
* **WordPress integration** - Seamless integration with your WordPress workflow

The plugin connects to the SEOForge MCP server to provide enterprise-grade SEO tools directly in your WordPress admin.

== Installation ==

= Automatic Installation =
1. Go to WordPress Admin → Plugins → Add New
2. Click "Upload Plugin"
3. Choose the seoforge-mcp-plugin.zip file
4. Click "Install Now"
5. Activate the plugin

= Manual Installation =
1. Upload the plugin files to `/wp-content/plugins/seoforge-mcp-plugin/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to SEOForge → Settings to configure the plugin

== Configuration ==

1. After activation, go to **SEOForge → Settings**
2. The plugin is pre-configured with the production API URL
3. Test the connection to ensure everything is working
4. Start using SEO features from the WordPress admin menu

== Usage ==

= Generating Content =
1. Go to **SEOForge → Generate Content**
2. Choose content type (blog post, article, product description)
3. Enter your topic and keywords
4. Select language (English or Thai)
5. Click "Generate" and review the AI-generated content

= SEO Analysis =
1. Edit any post or page
2. Use the SEOForge meta box to analyze content
3. Get real-time SEO recommendations
4. Optimize based on suggestions

= Keyword Research =
1. Go to **SEOForge → Keyword Research**
2. Enter your topic or industry
3. Get keyword suggestions with search volume data
4. Export keywords for content planning

== Frequently Asked Questions ==

= Is this plugin free? =
Yes, the plugin is free to use. It connects to the SEOForge MCP server which provides the AI-powered features.

= What languages are supported? =
Currently supports English and Thai content generation, with more languages planned for future releases.

= Do I need an API key? =
No API key is required. The plugin is pre-configured to work with the production SEOForge MCP server.

= Can I use this with any WordPress theme? =
Yes, the plugin works with any WordPress theme and integrates seamlessly with the WordPress admin interface.

= Is my content data secure? =
Yes, all communications with the MCP server are encrypted and secure. No content is stored permanently on external servers.
